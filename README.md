## 📁 Project:

- React
- Vite

## 📌 Dependencies

* ```react```
* ```react-dom```
* ```react-router-dom```
* ```tailwindcss```
* ```prop-types```
* ```@heroicons/react```

## 💼 Prerequisites

* Node.js
* npm

## ⚙️ Installation

* Install dependencies
    * ```npm install```
* Run the web server!
    * ```npm run dev```

## 🎈 Usage

* Navigate to http://localhost:5173 on your browser
